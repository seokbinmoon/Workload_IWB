# Workload_IWB (SEM)

This paper examines how perceived workload affects innovative work behavior through different pathways using structural equation modeling on 6,075 Korean civil servants. The results show that workload has a positive direct effect on innovative work behavior, increases both role stressors, and substantially reduces job autonomy. We also find that the negative indirect effect through job autonomy is the largest in magnitude, indicating that resource depletion is the dominant channel through which workload suppresses innovation.
